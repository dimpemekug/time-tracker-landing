# ⏱️ Time Tracker - App Landing Page

Una landing page moderna, minimale e responsiva per l'applicazione mobile **Time Tracker**. 
Progettata per presentare l'app con uno stile premium, include un mockup 3D interattivo, supporto completo per la Dark Mode e una struttura a griglia (Bento Grid).

## 🚀 Caratteristiche

*   **🎨 Design Moderno:** Stile ispirato ai trend attuali (Bento Grid, gradienti, glassmorphism).
*   **🌓 Dark Mode Completa:** Il sito rileva automaticamente la preferenza di sistema dell'utente. Include anche un toggle manuale (Sole/Luna) che salva la scelta nel browser.
*   **📱 3D Phone Mockup:** Un'animazione CSS avanzata che mostra l'interfaccia dell'app in 3D che ruota al passaggio del mouse. L'interfaccia dentro il telefono cambia colore in base al tema!
*   **⚡️ Performance:** Costruito con puro HTML e Tailwind CSS (via CDN). Nessun framework pesante, caricamento istantaneo.
*   **📱 Responsivo:** Ottimizzato perfettamente per Desktop, Tablet e Mobile.
*   **📄 Sezioni Legali Collassabili:** Privacy Policy e Supporto sono integrate a fondo pagina, con possibilità di aprire l'informativa completa in formato Markdown.

## 🛠️ Tecnologie Utilizzate

*   **HTML5** - Struttura semantica.
*   **Tailwind CSS (CDN)** - Styling e gestione del tema scuro.
*   **JavaScript (Vanilla)** - Logica per il cambio tema e salvataggio preferenze (`localStorage`).
*   **Google Fonts** - Font "Inter" per la tipografia.
